    一直想写点东西记录一下，眼瞅着三十了。尤其是哪些短暂出现又离开的人，曾经带给我短促的欢乐，又静悄悄的离开了，
想想总觉得记录下吧，往后的日子，回忆起来多少能够有个念头。
    今天刷到一个抖音，讲一个中学生，台风天，不用上学，没有作业，玩了一天，晚上妈妈又炖了汤喝。这种场景一下子，就让我带入了，
瞬间脑海里就是高一入学的第一天，萧瑟的秋天，毛毛细雨，到学校签到即可不用上课。路上各色的同学让我很新奇且满心期待、兴奋。
可能会遇见好看的女生做同学吧，也可能不会。可能遇见初中同学做同学吧，也可能不会。就是那种天然的期待，现在已经很少了，我多年的职场刻意
的训练就是让自己不那么情绪化，不那么怀有企图心和幻想。最近认识的一个女生过生日，看他好开心啊，于是我回复了她我也很开心，
可是，我知道，我只是陪衬一下当时的氛围，说了句应景的话，可是我并不是真的...
    之前一直念叨这要写回忆，今天算是个契机吧。想想，每个年龄段都有如此舒服美好的画面。小时候可能就是爸爸妈妈陪着我过生日，中学了可能就是喜欢的女孩子对我
微笑了下，大学可能就是狠女朋友在宾馆愉快的周末。虽然我是个很收敛、理性的boy，但难免有时有对这些瞬间有些动容。
    更多的，我是想记录我遇到的女生们和他们的情情爱爱。毕竟，作为一个男生，这是兴趣切入点，一点也不奇怪。
    之所以写在这里，毕竟做这行的么，歇着顺手。还有更重要的是，我既希望能够分享有趣的经历，写一些男男女女的事情，但又不想让太多人看到。
放这里最合适不过了。既希望有人能够体会到我的过往，又不希望能够打扰到我。倘若有一天这个star多的离谱，请不要让我上热搜，因为这既是我，其实也是你。
我在这里说的每句话都是真是的，所以不要放大到生活里去。
    还有，我不喜欢改错别字。
